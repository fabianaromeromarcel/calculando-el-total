Desafío Calculadora

Descripción

Este proyecto corresponde a un desafío de introducción a JavaScript. Consiste en una página web que muestra el precio base de un producto y permite aumentar o disminuir la cantidad mediante botones. El total a pagar se actualiza automáticamente según la cantidad seleccionada.

Funcionalidades

* Mostrar el precio base del producto.
* Incrementar la cantidad de productos.
* Disminuir la cantidad de productos.
* Calcular y mostrar el total a pagar en tiempo real.

Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

Aprendizajes

Durante este proyecto se practicaron conceptos como:

* Manipulación del DOM.
* Selección de elementos con querySelector.
* Modificación del contenido con innerHTML y textContent.
* Uso de variables.
* Operaciones aritméticas con JavaScript.
* Manejo de eventos mediante botones.
